# Submission assets

## Short description

Take my money turns unused time on a mobile-billed subscription into a safe, one-time credit for a new direct plan. GPT-5.6 reads synthetic receipt evidence and explains verified facts; deterministic code controls every financial and state-changing decision.

## Long description

Take my money is a subscription-portability prototype for customers who want to upgrade now without paying twice. A judge selects a synthetic receipt, GPT-5.6 extracts unverified evidence through Structured Outputs, and a provider-neutral sandbox adapter verifies the paid period and status. Bigint UTC arithmetic calculates the unused value, an idempotent saga consumes it once, and an append-only ledger records the simulated target-plan migration. GPT-5.6 then explains only server-verified facts behind a numerical grounding guard. The demo includes blocked, manual-review, AI-outage, excess-credit, and unknown-provider-result scenarios, plus a sanitized system audit. It never touches a real subscription, payment, refund, cancellation, or customer account.
